Estimar \(\theta\) por m√©todos Bayesianos#

Uma das desvantagens do m√©todo de infer√™ncia ML explicado acima √© que para candidatos com muito poucos ou muitos acertos as estimativas de \(\theta\) ficam inst√°veis. O ENEM usa um m√©todo de estima√ß√£o da nota chamado ‚ÄúExpected a Posteriori‚Äù (EAP), que usa o teorema de Bayes para inverter o modelo para a probabilidade das padr√µes e a habilidade \(\theta\):

\[ \text{prob}(\theta|\vec{x}) = \frac{\text{prob}(\vec{x}|\theta)\text{prob}(\theta)}{\text{prob}{(\vec{x})}} \]

Na pr√°tica, o termo \(\text{prob}{(\vec{x})}\) √© s√≥ um fator de normaliza√ß√£o para que o posterior \(\text{prob}(\theta|\vec{x})\) √© uma probabilidade. O que importa na express√£o acima √© a multiplica√ß√£o pelo ‚Äúprior‚Äù \(\text{prob}(\theta)\), uma premissa sobre como as habilidades s√£o distribu√≠das na popula√ß√£o. Multiplicando o prior com a modelo \(\text{prob}(\vec{x}|\theta)\) resulta no posterior que pode ser interpretado como a distribui√ß√£o da probabilidade de um certo \(\theta\), dado a evid√™ncia: o padr√£o de respostas \(\vec{x}\).

Tendo o posterior para \(\theta\), podemos resumir o resultado de v√°rias formas. A estimativa EAP √© o valor esperado do \(\theta\)

\[ \theta_{\text{EAP}} = \int \theta \text{prob}(\theta|\vec{x}) d\theta \]

e uma estimativa para sua incerteza √© o desvio padr√£o do posterior.

No c√°lculo da nota do ENEM o prior √© uma normal com desvio padr√£o 1. Calcular o posterior √© surpreendemente simples na pr√°tica. √â somente usar o c√°lculo da verossimilhan√ßa que fizeram acima, multiplacado pela fun√ß√£o gaussiana \(N(0,1)\). O resultado n√£o √© normalizado (n√£o √© uma probabilidade), mas para fins de c√°lculo do valor esperado podemos simplesmente usar a posterior n√£o-normalizada como peso.

params = params.join(padr.reset_index().T).rename(columns={0:'acertou'})
theta = np.arange(-3,3,0.01)

def P(theta,a=1,b=0,c=0):
    'O modelo 3PL'
    return c + (1-c)*1/(1+np.exp(a*(b-theta)))


lh = np.ones(len(theta))
for item,value in params.iterrows():    
    acertou = int(value.iloc[-1])
    if acertou:
        lh = lh*P(theta,a=value['a_inep'],b=value['b_inep'],c=value['c_inep'])
    else:
        lh = lh*(1-P(theta,a=value['a_inep'],b=value['b_inep'],c=value['c_inep']))

    nota_ml = (-3+lh.argmax()*0.01)
    posterior = lh*norm.pdf(theta)
    nota_eap = np.average(theta, weights=posterior)
